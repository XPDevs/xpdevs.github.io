DoorsOS

DoorsOS is an experimental web-based operating system designed for simplicity, efficiency, and innovation.

  • Lightweight Design: Minimalist approach ensures fast performance.
  • Customizable Interface: Adjust themes and layouts easily.
  • Web-Based: Runs directly in your web browser.
  • Open Source: Available on GitHub for contributions.
