Buttons


3x3 Buttons 


pushd "%~dp0..\System"
Set "Path=%cd%;%cd%\Files;%Path%;"
Popd

Color 80
Call Button3 5 3 08 "button" 15 3 08 "button" 30 3 08 "button" X _Box _hover _Code

:Loop
REM To reprint the buttons Quick, without re-calculating every button each time...
REM This SAVEs a lot of time...
Batbox %_Code%

REM To check, if a Button is clicked or not.
GetInput /M %_Box% /H %_hover%
if %errorlevel% equ 1 button
if %errorlevel% equ 2 goto :button
if %errorlevel% equ 3 goto :button
